Core Cycle Steps:
1. Glc₃Man₉GlcNAc₂ N-glycan attached → glucosidase I/II → Glc₁Man₉GlcNAc₂
2. CNX/CRT binds monoglucosylated glycan → recruits ERp57
3. ERp57 oxidizes/isomerizes disulfides in substrate
4. Glucosidase II removes final glucose → release from CNX/CRT
5. Correctly folded → Golgi; misfolded → UGGT reglucosylates → cycle repeats
6. Persistent misfolding → mannose trimming → ERAD

Key Players:
- ERp57 (PDIA3): 4 Trx domains, CGHC sites, lectin-binding to CNX/CRT
- CNX (membrane) / CRT (soluble): Lectin chaperones
- UGGT: Folding sensor → reglucosylation
- Glucosidase II: Release trigger
- ER mannosidase I: ERAD signal

Relevance to Keratin:
- α-Keratin: CNX/CRT/ERp57 ensure correct inter-chain disulfides in IF assembly
- β-Keratin: Less N-glycosylation, but PDI family (ERp57) aids cross-links
